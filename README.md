# Official Homebrew Tap for AI Router

Install [AI Router](https://github.com/soooooollee/ai-router) with Homebrew:

```bash
brew install soooooollee/ai-router/airoute
air init
air start
```

Update AI Router with:

```bash
brew update
brew upgrade airoute
air restart
```

This repository is the project-specific Homebrew package source for AI Router. The formula is updated automatically when a new AI Router GitHub Release is published.
